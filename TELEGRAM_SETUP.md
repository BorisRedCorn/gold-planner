# Gold Planner → Telegram: пошаговая инструкция

Чтобы **любой человек** мог открыть бота и пользоваться **своими личными задачами**, нужно сделать 4 вещи: выложить сайт в интернет, создать бота, связать их и поделиться ссылкой.

> **Важно:** Telegram принимает Mini App только по **HTTPS** (обычный `localhost` не подойдёт для других людей).

---

## Шаг 1. Зарегистрируйтесь на GitHub

1. Откройте [github.com](https://github.com) и создайте аккаунт (если ещё нет).
2. Нажмите **+** → **New repository**.
3. Имя: `gold-planner`, тип **Public**, создайте репозиторий.

---

## Шаг 2. Загрузите проект на GitHub

Откройте терминал в папке проекта (`Gold_Planner`) и выполните:

```bash
git init
git add .
git commit -m "Gold Planner Telegram Mini App"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/gold-planner.git
git push -u origin main
```

Замените `ВАШ_ЛОГИН` на свой логин GitHub.

> Если Git спросит логин/пароль — используйте **Personal Access Token** вместо пароля (GitHub → Settings → Developer settings → Tokens).

---

## Шаг 3. Задеплойте приложение на Vercel (бесплатно)

1. Откройте [vercel.com](https://vercel.com) → войдите через GitHub.
2. **Add New… → Project**.
3. Выберите репозиторий `gold-planner`.
4. Настройки (обычно подставляются сами):
   - **Framework Preset:** Vite
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
5. Нажмите **Deploy**.
6. Через 1–2 минуты получите ссылку, например:
   ```
   https://gold-planner-abc123.vercel.app
   ```
   **Скопируйте её** — это адрес вашего приложения.

---

## Шаг 4. Создайте Telegram-бота

1. В Telegram найдите **@BotFather**.
2. Отправьте: `/newbot`
3. Придумайте имя: `Gold Planner`
4. Придумайте username (должен заканчиваться на `bot`), например: `gold_planner_bubzik_bot`
5. BotFather пришлёт **токен** вида:
   ```
   7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
   **Сохраните его** — никому не показывайте.

---

## Шаг 5. Привяжите домен к боту

BotFather должен знать, с какого сайта открывается приложение.

1. В чате с **@BotFather** отправьте: `/setdomain`
2. Выберите своего бота.
3. Введите **только домен** (без `https://`), например:
   ```
   gold-planner-abc123.vercel.app
   ```

---

## Шаг 6. Настройте кнопку меню (главный способ открытия)

1. В **@BotFather** отправьте: `/setmenubutton`
2. Выберите своего бота.
3. Выберите **Configure menu button**.
4. Текст кнопки: `Gold Planner`
5. URL — **полная ссылка** с Vercel:
   ```
   https://gold-planner-abc123.vercel.app
   ```

Теперь у каждого пользователя внизу чата с ботом появится кнопка **«Gold Planner»** — приложение открывается одним нажатием.

---

## Шаг 7. (Опционально) Запустите бота для команды /start

Если хотите, чтобы бот отвечал на `/start` с красивой кнопкой:

1. В папке проекта создайте файл `.env`:

```env
BOT_TOKEN=7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
WEBAPP_URL=https://gold-planner-abc123.vercel.app
```

2. Запустите:

```bash
npm run bot
```

3. Напишите боту `/start` — появится кнопка **«✦ Открыть Gold Planner»**.

> Бот должен быть **запущен на компьютере или сервере**, чтобы отвечать на сообщения. Кнопка меню из шага 6 работает **без** запущенного бота.

---

## Шаг 8. Поделитесь ботом с людьми

Отправьте друзьям ссылку на бота:

```
https://t.me/ВАШ_USERNAME_БОТА
```

Например: `https://t.me/gold_planner_bubzik_bot`

### Что делает каждый пользователь:

1. Переходит по ссылке → **Start** / **Запустить**
2. Нажимает кнопку **Gold Planner** внизу (или `/start`)
3. Открывается приложение — **у каждого свои задачи**, они не видят чужие

---

## Как работает «личное» использование

| Что | Как |
|-----|-----|
| Задачи каждого человека | Хранятся отдельно по Telegram ID |
| Чужие задачи | Не видны |
| Сохранение | На телефоне пользователя (localStorage) |
| Первый запуск | Стартовый набор задач ресторана, дальше — только свои изменения |

---

## Обновление приложения

После изменений в коде:

```bash
git add .
git commit -m "обновление"
git push
```

Vercel **сам пересоберёт** сайт за 1–2 минуты. Бот и BotFather менять не нужно.

---

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| «Приложение не открывается» | Проверьте `/setdomain` и URL в `/setmenubutton` |
| «Открывается пустая страница» | Подождите деплой на Vercel, обновите страницу |
| «Бот не отвечает на /start» | Запустите `npm run bot` и проверьте `.env` |
| «Invalid bot token» | Скопируйте токен заново из BotFather |
